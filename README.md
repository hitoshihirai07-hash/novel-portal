# Novel Portal（最小構成）

- タイトル固定：Novel Portal
- 公開：トップで検索→目次→読む
- 投稿：投稿ページ
- マイページ：自分の作品一覧・検索
- 管理：管理ページ（トップに表示しない運用）

---

## 運用メモ（管理者向け）
- D1：`novel-4`（Pages の D1 binding で `DB` に紐付け）
- 環境変数：`INVITE_CODE`, `ADMIN_TOKEN`
- テーブル作成は初回アクセス時に自動作成（Functions 側で CREATE TABLE IF NOT EXISTS）


## 18歳以上向け
- 投稿時に「18歳以上向け」をチェックできます。
- 読む時に年齢確認ポップアップが出ます（作品ごとに1回、端末に保存）。


## 管理（追加）
- 管理ページで作品検索（本文検索/フィルタ）
- 作者一覧（無効化：作品は全て非表示）
- 監査ログ（管理操作）
- エクスポート（JSON）
